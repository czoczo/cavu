<Steps>
1. Make sure you have [docker](https://docs.docker.com/engine/install/) installed.

2. Download source code from GitHub. There are usually three methods to choose from. Use the one that suits you best.
   ```sh
   # HTTPS:
   git clone https://github.com/czoczo/cavu.git
   
   # SSH:
   git clone git@github.com:czoczo/cavu.git
   
   # GitHub CLI:
   gh repo clone czoczo/cavu
   ```
   
3. Build contianer
   ```sh
   docker buildx build \
     --platform linux/amd64,linux/arm64 \
     -t casavue:latest .
   ```
</Steps>
